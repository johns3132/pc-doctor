PC Doctor

Webowa aplikacja do monitorowania podstawowych parametrów systemu:
- CPU
- RAM
- Dysk

Technologie
- Python (Flask)
- HTML / CSS / JavaScript

Uruchomienie lokalne

```bash
git clone https://github.com/johns3132/pc-doctor.git
cd pc-doctor
python3 -m venv venv
source venv/bin/activate   # Windows: venv\Scripts\activate
pip install -r requirements.txt
python app.py
